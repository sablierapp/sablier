Sablier follows [Semantic Versioning 2.0.0](https://semver.org/) (SemVer). Given a version number `MAJOR.MINOR.PATCH`:

- **MAJOR**: incompatible API changes
- **MINOR**: backwards-compatible new functionality
- **PATCH**: backwards-compatible bug fixes

Releases are cut automatically from [Conventional Commits](https://www.conventionalcommits.org/) by [release-please](https://github.com/googleapis/release-please).
